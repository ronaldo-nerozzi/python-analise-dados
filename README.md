# 🐍 Python — Análise Exploratória de Dados

Repositório com scripts e notebooks Python para análise exploratória de dados, com foco em extração de insights, visualizações e tratamento de dados.

---

## 🎯 Objetivo

Demonstrar habilidades práticas em Python para dados, incluindo:
- Análise exploratória de dados (EDA)
- Limpeza e tratamento de dados com Pandas
- Visualizações com Matplotlib e Seaborn
- Leitura e manipulação de arquivos CSV e Excel
- Estatísticas descritivas

---

## 📁 Projetos

### 01 — Análise de Fundos de Investimento CVM *(em breve)*
Análise exploratória dos dados públicos de fundos brasileiros: patrimônio, captação e evolução temporal.

**Bibliotecas:** Pandas, Matplotlib, Seaborn

---

### 02 — Análise do Mercado Financeiro Brasileiro *(em breve)*
Exploração de indicadores econômicos como IPCA, SELIC e CDI ao longo do tempo.

**Bibliotecas:** Pandas, Matplotlib, Requests

## 📌 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/ronaldo-nerozzi/python-analise-dados.git
```
2. Instale as dependências:
```bash
pip install pandas matplotlib seaborn jupyter
```
3. Abra o notebook desejado no Jupyter ou VSCode
